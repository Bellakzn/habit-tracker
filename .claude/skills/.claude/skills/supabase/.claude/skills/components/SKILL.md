# UI Components Skill

## Triggers
- "создай компонент"
- "добавь кнопку"
- "сделай форму"
- "create component"
- "UI"

## Description
Правила создания UI компонентов для проекта.

## Rules
- Все стили через Tailwind классы
- Никаких inline style={{}}
- Никаких className пропсов снаружи
- Визуальные варианты через пропс variant
- Компонент должен быть самодостаточным

## Component structure
- Button: variant (primary/secondary/ghost), size (sm/md/lg), disabled
- Input: label, placeholder, error, type
- Card: children, padding
- Badge: variant (success/warning/error/default), label
